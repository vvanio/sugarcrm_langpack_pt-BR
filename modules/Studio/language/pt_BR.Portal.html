Studio

Voce pode escolher uma das seguintes opções para editar o SugarPortal:

  • Editar um módulo. Selecione esta opção para editar os módulos de Bug Tracker, Ocorrências ou Leads
  • Carregar uma Folha de Estilos. Selecione esta opção para carregar uma stylesheet no SugarPortal.
  • Sincronizar o Portal. Selecione esta opção para sincronizar o Portal com o Sugar.

Para editar um módulo, faça o seguinte:

  1. Selecione o módulo que deseja reconfigurar no SugarPortal.
  2. Selecione a tela que você deseja alterar.
    O Editor do Portal vai exibir uma barra de ferramentas no lado esquerdo e os campos de detalhe à direita. Para alterar a Tela de Edição ou Listagem, Selecione o link apropriado na parte superior da tela. A barra de ferramentas vai exibir os campos disponíveis que atualmente não estão na tela de detalhe. Também exibe um campo vazio e um campo em branco. Utilize o campo em branco para criar espaços na tela. Por exemplo, use o campo em branco ao lado da Descrição para reservar mais espaço para uma descrição detalhada. Use um campo vazio para reservar espaço no formulário para o futuro.
  3. Para acrescentar um campo da barra de ferramentas na tela de detalhe, clique em Adicionar Linha. Selecione o campo da barra de ferramentas e arraste-o para o campo vazio.
  4. Para remover um campo da tela, selecione-o, e arraste-o para a barra de ferramentas.
  5. Clique Salvar para salvar as suas alterações.

Para carregar uma style sheet, faça o seguinte:

  1. Selecione a aba apropriada para os módulos de Ocorrências, Bugs, ou Newsletters.
  2. Clique Procurar, navegue até a localização da style sheet, e selecione-a.
  3. Clique Upload para carregar a style sheet.

Para sincronizar o Portal com o servidor Sugar, faça o seguinte:

  1. Informe a URL da instância do portal que quer atualizar e clique em Avançar.
  2. Insira o seu nome de usuário e senha e clique em Iniciar Sincronização.
